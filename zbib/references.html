7. Citing SumTimes¶

SumTimes relies on the DendroPy Phylogenetic Computing Library. Accordingly [1], please cite DendroPy as:

Sukumaran, J. and M. T. Holder. 2010. DendroPy: a Python library for phylogenetic computing. Bioinformatics 26: 1569-1571.

8. Literature Cited¶

[1]Jeet Sukumaran and Mark T. Holder. DendroPy: a Python library for phylogenetic computing. Bioinformatics, 26:1569–1571, 2010.
